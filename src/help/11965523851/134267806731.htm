Narzędzia -> Transfer -> Ustawienia połączenia z chmurą

LOGO!Soft Comfort wymaga potwierdzenia interfejsu połączenia dla każdej próby komunikacji. Zobacz "Narzędzia -> Transfer" aby uzyskać pełne informacje.

Za pomocą tego polecenia menu można skonfigurować połączenie z chmurą dla LOGO! BM.

Uwaga

Czas LOGO! BM

Przed skonfigurowaniem połączenia z chmurą upewnij się, że data i czas LOGO! BM są poprawne. W przeciwnym razie połączenie z chmurą nie powiedzie się.

Uwaga

LOGO! 8.4 (i nowsze wersje) Base Module obsługuje połączenie z chmurą AWS, Azure, Alibaba lub ogólnym brokerem MQTT.

LOGO! 8.3 Base Module obsługuje tylko połączenie z chmurą AWS.

Włącz dostęp do chmury

Dostęp do chmury można włączyć w następujący sposób:

  1. Zaznacz pole wyboru "Włącz dostęp do chmury".

    Uwaga

    • Dla LOGO! 8.3 Base Module nie można jednocześnie włączyć dostępu do serwera WWW i dostępu do chmury.

    • Dla LOGO! 8.4 Base Module można jednocześnie włączyć dostęp do serwera WWW i dostęp do chmury.

  2. Kliknij przycisk "Zastosuj".

Zarejestruj LOGO! BM w AWS IoT

Wymagania

Przed zarejestrowaniem LOGO! BM w AWS IoT upewnij się, że spełnione są następujące warunki:

  • Dostęp do chmury musi być włączony.

  • Konto AWS zostało utworzone.

  • Zastosowano klucz dostępu.

Procedura

Wykonaj następujące kroki, aby zarejestrować LOGO! BM w AWS IoT:

  1. Przejdź do zakładki "Narzędzia -> Transfer -> Ustawienia połączenia z chmurą" i kliknij przycisk "Zarejestruj urządzenie". Następnie wyświetli się okno dialogowe "Rejestracja".

    Wybierz typ chmury "AWS" i kliknij przycisk "Dalej".

  2. Wybierz region IoT, wprowadź "ID klucza dostępu" i "Sekretny klucz dostępu". Jeśli używasz konta AWS z tymczasowymi poświadczeniami (TVM) do rejestracji, musisz również wprowadzić "Token sesji". Następnie kliknij przycisk "Dalej".

  3. Zaznacz pole wyboru "Utwórz nowe urządzenie" lub "Użyj istniejącego urządzenia". Następnie kliknij przycisk "Dalej".

    Jeśli wybierzesz "Utwórz nowe urządzenie", przejdź do kroku 4. Jeśli wybierzesz "Użyj istniejącego urządzenia", przejdź do kroku 5.

  4. Wprowadź nazwę. Następnie kliknij przycisk "Dalej".

    Zasady nazewnictwa:

    • Długość nazwy nie może przekraczać 128 znaków.

    • Nazwa może składać się z dowolnych liter, myślnika "-" i podkreślnika "_".

    Więcej informacji można znaleźć w zasadach nazewnictwa urządzeń AWS IoT.

    Po utworzeniu grupy rozliczeniowej i przypisaniu do niej tagów na stronie AWS, można przypisać skonfigurowaną grupę rozliczeniową do urządzenia IoT, aby kategoryzować i śledzić koszty związane z usługą IoT.

  5. Wybierz typ tematu dla rejestracji. Następnie kliknij przycisk "Dalej".

    Jeśli zarejestrowane urządzenie IoT będzie używane w projekcie LWE, który zostanie wdrożony w AWS jako aplikacja internetowa, należy wybrać temat Shadow i skonfigurować dane chmury jako "Format danych serwera WWW LOGO!" w Ustawieniach transferu danych chmury.

    Uwaga

    Dla LOGO! 8.3 Base Module nie trzeba wybierać typu tematu, ale można przejść bezpośrednio do kroku 6.

  6. Utwórz certyfikat, wprowadzając następujące informacje:

    Element

    Zasada

    Kraj/Region

    Wybierz z listy rozwijanej.

    Województwo

    Zasady nazewnictwa:

    • Długość nazwy województwa lub organizacji nie może przekraczać 16 znaków.

    • Nazwa województwa lub organizacji może składać się z dowolnych znaków alfanumerycznych, spacji, myślnika "-", podkreślnika "_" i dwukropka ":".

    Wiêcej informacji można znaleźć w zasadach dotyczących usługi AWS IoT.

    Organizacja

    Nazwa jednostki

    Zasady nazewnictwa:

    • Długość nazwy jednostki nie może przekraczać 16 znaków.

    • Nazwa jednostki może składać się z dowolnych znaków alfanumerycznych.

    Wiêcej informacji można znaleźć w zasadach dotyczących usługi AWS IoT.

  7. Kliknij przycisk "Dalej", a następnie na stronie "Status rejestracji" zostanie wyświetlony komunikat o pomyślnej rejestracji. Kliknij przycisk "Zakończ", aby zamknąć okno dialogowe rejestracji.

Zarejestruj LOGO! BM w Azure IoT

Wymagania

Przed zarejestrowaniem LOGO! BM w Azure IoT upewnij się, że spełnione są następujące warunki:

  • Dostęp do chmury musi być włączony.

  • Konto Azure zostało utworzone.

  • Zastosowano podstawowy ciąg połączenia IoT Hub.

Procedura

Wykonaj następujące kroki, aby zarejestrować LOGO! BM w Azure IoT:

  1. Przejdź do zakładki "Narzędzia -> Transfer -> Ustawienia połączenia z chmurą" i kliknij przycisk "Zarejestruj urządzenie". Następnie wyświetli się okno dialogowe "Rejestracja".

    Wybierz typ chmury "Azure" i kliknij przycisk "Dalej".

  2. Wprowadź "Podstawowy ciąg połączenia IoT Hub". Następnie kliknij przycisk "Dalej".

  3. Wybierz "Utwórz nowe urządzenie" lub "Użyj istniejącego urządzenia". Następnie kliknij przycisk "Dalej".

    Jeśli wybierzesz "Utwórz nowe urządzenie", przejdź do kroku 4; jeśli wybierzesz "Użyj istniejącego urządzenia", przejdź do kroku 5.

  4. Wprowadź identyfikator urządzenia. Następnie kliknij przycisk "Dalej".

    Zasady nazewnictwa:

    • Długość identyfikatora urządzenia nie może przekraczać 128 znaków.

    • Identyfikator urządzenia może składać się z dowolnych liter, cyfr i niektórych znaków specjalnych - . _ * ! ( ) , : = @ $ '.

    Uwaga

    Jeśli utworzysz identyfikator urządzenia na platformie Azure z innymi znakami specjalnymi niż wymienione powyżej, nie będzie on wyświetlany na liście istniejących urządzeń w kroku 3.

  5. Na stronie "Status rejestracji" zostanie wyświetlony komunikat o pomyślnej rejestracji. Kliknij przycisk "Zakończ", aby zamknąć okno dialogowe rejestracji.

Narzędzia -> Transfer -> Dynamiczny filtr IP serwera (tylko wersje 0BA8 i nowsze)

LOGO!Soft Comfort wymaga potwierdzenia interfejsu połączenia przy każdej próbie komunikacji. Pełne informacje znajdziesz w "Narzędzia -> Transfer".

Użyj tego polecenia menu, aby wyświetlić i skonfigurować ACL (Access Control List) podłączonego modułu bazowego LOGO!:

Jeśli zaznaczysz pole "①", moduł bazowy LOGO! zezwala na żądania dostępu ze wszystkich adresów IP.

Jeśli zaznaczysz pole "②", LOGO!Soft Comfort udostępnia listę dozwolonych adresów IP z ośmioma polami wyboru, każdy z polem wprowadzania adresu IP do konfiguracji filtrowania IP.

Kliknij pole wyboru na liście, wprowadź adres IP modułu bazowego LOGO! i kliknij "③", aby autoryzować jego dostęp:

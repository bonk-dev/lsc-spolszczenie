Narzędzia -> Wybierz sprzęt

Masz dwie możliwości wyboru urządzenia podczas tworzenia programu lub projektu w LOGO!Soft Comfort:

  • Możesz najpierw utworzyć program lub projekt, a następnie określić wymagane LOGO! za pomocą okna dialogowego Narzędzia → Określ LOGO!.

  • Możesz najpierw określić wersję LOGO!, dla której chcesz utworzyć program obwodów lub projekt, wywołując okno dialogowe Narzędzia → Wybierz sprzęt. Urządzenie można również wybrać, klikając dwukrotnie ikonę LOGO! na pasku stanu.

Okno dialogowe wyboru urządzenia pokazuje wszystkie dostępne bloki i zasoby pamięci.

Jeśli Twój program zawiera określone bloki ze specjalnymi funkcjonalnościami, okno dialogowe wyboru sprzętu oferuje tylko urządzenia LOGO! zgodne z tymi blokami.

Jeśli pracujesz nad projektem sieciowym, okno dialogowe wyboru sprzętu oferuje tylko urządzenia LOGO! obsługujące połączenia Ethernet.

Zawartość DVD

DVD-ROM

DVD-ROM zawiera oprogramowanie instalacyjne LOGO!Soft Comfort oraz dodatkowe przydatne informacje.

Start.html

Ten plik przeprowadzi Cię przez zawartość DVD-ROM. Pomaga w wykonaniu następujących zadań:

  • Zainstaluj LOGO!Soft Comfort

  • Uruchom LOGO!Soft Comfort

Plik zapewnia również dostęp do następujących elementów:

  • Rysunki CAD

  • Instrukcje

  • Sterowniki

Foldery na DVD-ROM:

Poniższa lista opisuje zawartość niektórych folderów na DVD-ROM:

  • Folder ..\CAD zawiera rysunki CAD urządzeń LOGO!.

  • Folder ..\Manual zawiera aktualną Instrukcję LOGO! w formacie PDF, którą można odczytać za pomocą Acrobat Reader.

  • Folder ..\Onlinehelp zawiera aktualną pomoc online LOGO!Soft Comfort w formacie PDF, którą można odczytać za pomocą Acrobat Reader.

  • Folder ..\Tools zawiera następujące trzy foldery i plik resetujący:

    • Folder ..\LOGO! Access Tool zawiera dodatek do Excela oraz aktualną pomoc LOGO! Access Tool w formacie PDF, którą można odczytać za pomocą Acrobat Reader.

    • Folder ..\LOGO! Web Editor Tool zawiera plik instalacyjny oraz aktualną pomoc online LOGO! Web Editor w formacie PDF, którą można odczytać za pomocą Acrobat Reader.

    • Folder ..\LOGO! Integrity Tool zawiera plik .exe oraz aktualny plik Readme LOGO! Integrity Tool w formacie PDF, którą można odczytać za pomocą Acrobat Reader.

    • Plik ...\_reset.bm służy do przywrócenia ustawień fabrycznych LOGO! BM. Szczegółowe instrukcje znajdziesz w Przywracanie ustawień fabrycznych LOGO!.

  • Folder..\Readme zawiera plik readme w formacie HTML z informacjami, które należy znać przed instalacją.

  • Folder ..\Sample zawiera kilka przykładowych aplikacji, które przedstawiają przykładowe rozwiązania dla wielu zastosowań wszechstronnego LOGO!.

  • DVD-ROM zawiera również wersje instalacyjne LOGO!Soft Comfort dla różnych systemów operacyjnych:

    • Folder ..\Linux\Application_64 zawiera pliki instalacyjne dla 64-bitowych systemów operacyjnych Linux.

    • Folder ..\Windows\Application_32 zawiera pliki instalacyjne dla 32-bitowych systemów operacyjnych Windows.

    • Folder ..\Windows\Application_64 zawiera pliki instalacyjne dla 64-bitowych systemów operacyjnych Windows.

    • Folder ...\Mac\Application zawiera pliki instalacyjne dla systemów operacyjnych Apple Macintosh.

Wystarczy skopiować odpowiedni folder ..\Application na dysk twardy i uruchomić LOGO!Soft Comfort, wywołując ..\Application\LOGOComfort.

Folder ..\Windows\ Application_32 jest przeznaczony dla 32-bitowych systemów operacyjnych Windows, a folder ..\Windows\ Application_64 dla 64-bitowych systemów operacyjnych Windows.

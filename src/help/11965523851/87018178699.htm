Importowanie urządzenia

Możesz zaimportować program obwodu i jego konfiguracje sieciowe do widoku sieci.

Importowanie programu obwodu do urządzenia LOGO!

Wykonaj poniższe kroki, aby zaimportować obwód do swojego projektu.

  1. Kliknij prawym przyciskiem myszy na urządzenie w widoku sieci, a następnie kliknij ①.

  2. Wybierz program obwodu, który chcesz zaimportować na swoim komputerze.

Uwaga

Możesz zaimportować swój program obwodu tylko do urządzenia LOGO! 0BA7 i nowszych wersji.
Uwaga

Importowany program obwodu zastępuje poprzedni program obwodu zapisany w urządzeniu LOGO!.

Gdy importujesz program obwodu do urządzenia, które ma inną wersję sprzętową, LOGO!Soft Comfort zsynchronizuje ich wersję sprzętową zgodnie z ich własnym stanem. Dla programów obwodu, które nie obsługują funkcji sieciowych, nie można ich zaimportować do urządzenia. Szczegółowe informacje znajdziesz na poniższym schemacie blokowym.

Tworzenie nowego urządzenia LOGO! 8.1 & 8.2 (LOGO! 8.FS4) przez import programu obwodu

Możesz utworzyć nowe urządzenie LOGO! 8.1 & 8.2 (LOGO! 8.FS4) na jeden z poniższych sposobów.

  • Import w widoku sieci

    • Kliknij prawym przyciskiem myszy na dowolne puste miejsce w widoku sieci, a następnie kliknij ①.

    • Wybierz program obwodu, który chcesz zaimportować na swoim komputerze.

  • Przeciągnij i upuść program obwodu do drzewa projektu sieci.

Odzyskiwanie połączenia sieciowego

Podczas importowania LSC sprawdza adres IP importowanego urządzenia. Jeśli adres IP istnieje w bieżącym projekcie sieci, LSC informuje, że adres IP i konfiguracja sieciowa wymagają aktualizacji. Po kliknięciu Tak, LSC przypisuje nowy adres IP importowanemu urządzeniu.

LSC ponownie łączy złącza, gdy ich następująca konfiguracja jest zgodna:

  • Klient i serwer.

  • Adres IP

  • TSAP

Więcej informacji na temat ustawień sieciowych znajdziesz w Tworzenie połączenia Ethernet.

Uwaga

Gdy dynamiczny port urządzenia jest całkowicie zajęty, połączenie z tym portem nie może być już odzyskane.

Narzędzia -> Transfer -> Wyczyść program użytkownika i hasło

LOGO!Soft Comfort wymaga potwierdzenia interfejsu połączenia przy każdej próbie komunikacji. Szczegółowe informacje znajdziesz w "Narzędzia -> Transfer".

To polecenie pozwala wyczyścić program obwodu w podłączonym urządzeniu LOGO! oraz hasło programu, jeśli istnieje.

Musisz odpowiedzieć na okno dialogowe potwierdzenia, aby upewnić się, że naprawdę chcesz wyczyścić zarówno program obwodu, jak i hasło (jeśli jest skonfigurowane) z urządzenia LOGO!. Po potwierdzeniu polecenia, LOGO!Soft Comfort wykonuje operację czyszczenia. Jeśli nie potwierdzisz polecenia, LOGO!Soft Comfort nie podejmie żadnych działań. Program obwodu i hasło (jeśli skonfigurowane) pozostaną w urządzeniu LOGO!.

Uwaga

Urządzenia LOGO! starsze niż wersja 0BA6 nie obsługują tej funkcji. Jeśli spróbujesz wykonać to polecenie na starszym urządzeniu, LOGO!Soft Comfort wyświetli komunikat informujący, że urządzenie nie obsługuje tej funkcji.

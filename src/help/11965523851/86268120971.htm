Narzędzia -> Transfer -> Ustawienia NTP (tylko LOGO! 8.FS4 i nowsze wersje)

LOGO!Soft Comfort wymaga potwierdzenia interfejsu połączenia przy każdej próbie komunikacji. Zobacz "Narzędzia -> Transfer" aby uzyskać pełne informacje.

To polecenie pozwala skonfigurować ustawienia NTP (Network Time Protocol). Za jego pomocą można ustawić BM jako serwer lub klient NTP. Funkcja NTP jest wyłączona, jeśli żaden tryb nie jest wybrany.

Jeśli włączysz funkcję NTP dla modułu bazowego LOGO!, musisz ustawić dla niego strefę czasową.

Jeśli ustawisz moduł bazowy LOGO! jako klient NTP, musisz wprowadzić adres IP serwera NTP dla modułu.

Uwaga

Obecnie LOGO! może rozpoznawać tylko adresy IP, a nie nazwy hostów. Proszę najpierw przekształcić nazwę hosta serwera NTP na adres IP.

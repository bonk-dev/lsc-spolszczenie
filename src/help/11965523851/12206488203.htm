Klawisze funkcyjne i skróty

LOGO!Soft Comfort oferuje następujące klawisze funkcyjne i skróty dla często używanych funkcji:

Klawisze funkcyjne w LOGO!Soft Comfort:

[F1]

Wywołuje kontekstową Pomoc online

[F2]

Narzędzia → Określ LOGO!

[F3]

Uruchomienie/zakończenie symulacji

[F4]

Widok → Okno informacyjne

[F5]

Narzędzie łączenia

[F6]

Narzędzie stałych i złączy

[F7]

Narzędzie podstawowych funkcji

[F8]

Narzędzie specjalnych funkcji

[F9]

Narzędzie tekstu/Narzędzie wstawiania komentarzy

[F10]

Otwiera pasek menu

[F11]

Narzędzie cięcia/łączenia

Skróty w LOGO!Soft Comfort:

 

W menu Plik :

[Ctrl+N]

Plik → Nowy (otwiera domyślny edytor określony w Narzędzia/Opcje/Edytor)

[Ctrl+O]

Plik → Otwórz

[Ctrl+F4]

Plik → Zamknij

[Ctrl+S]

Plik → Zapisz

[Ctrl+Shift+F1]

Plik → Podgląd wydruku

[Ctrl+P]

Plik → Drukuj

[Alt+Enter]

Plik → Właściwości

[Alt+F4]

Plik → Zakończ

 

W menu Edycja :

[Ctrl+Z]

Edycja → Cofnij

[Ctrl+Y]

Edycja → Ponów

[DELETE]

Edycja → Usuń

[Ctrl+X]

Edycja → Wytnij

[Ctrl+C]

Edycja → Kopiuj

[Ctrl+V]

Edycja → Wklej

[Ctrl+A]

Edycja → Zaznacz wszystko

[Ctrl+G]

Edycja → Przejdź do bloku

 

W menu Widok :

[Ctrl+M]

Wybierz połączenia

[Ctrl+mouse wheel]

Widok → Powiększ

Widok → Pomniejsz

 

W menu Narzędzia :

[Ctrl+D]

Narzędzia → Transfer: PC → LOGO!

[Ctrl+U]

Narzędzia → Transfer: LOGO! → PC

[Ctrl+H]

Narzędzia → Wybierz sprzęt

[Ctrl+-]

Narzędzia → Porównaj

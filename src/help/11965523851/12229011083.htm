Ograniczenie dla rodziny urządzeń 0BA4

Wzmocnienie nie może być wartością ujemną.

Obliczenia dla rodzin urządzeń 0BA0 do 0BA3

W przypadku urządzeń LOGO! z tych rodzin, LOGO! dodaje lub odejmuje parametr offset od znormalizowanej wartości przed pomnożeniem wartości przez parametr wzmocnienia.

Dlatego obowiązują następujące wzory:

Wartość analogowa = (wartość znormalizowana + offset) x (wzmocnienie x 100)

Wzmocnienie (w procentach) = (maxCzujnik - minCzujnik) / [(maxnorm – minnorm) x 100]

Offset = [(minCzujnik x maxnorm) – (maxCzujnik x minnorm)] / (maxCzujnik – minCzujnik)

Wzmocnienie (w procentach) = minCzujnik / [(minnorm + offset) x 100]

Offset = [maxCzujnik / (wzmocnienie x 100)] - maxnorm

Wzmocnienie

Ten parametr jest podawany w %.

Wzmocnienie nie może być wartością ujemną.

Przesunięcie punktu zerowego

Można wprowadzić wartości od -999 do +999 dla przesunięcia punktu zerowego.

Narzędzia -> Transfer -> Ustawienia transferu danych w chmurze

LOGO!Soft Comfort wymaga potwierdzenia interfejsu połączenia dla każdej próby komunikacji. Zobacz "Narzędzia -> Transfer" aby uzyskać pełne informacje.

To menu pozwala skonfigurować transfer danych w chmurze.

Poniższa tabela zawiera definicje dla każdej pozycji:

Pozycja

Definicja

ID

Uwaga: Maksymalna liczba rekordów danych wynosi 64.

Nazwa

Nazwa danych w chmurze.

Uwaga: Nazwa może zawierać litery, cyfry, znaki specjalne "-" i "_", a maksymalny rozmiar nazwy to 15 bajtów.

Zakres

Obszar pamięci w LOGO! BM.

Adres

Adres pamięci.

Typ

Typ danych.

Długość

Długość danych.

Z częstotliwością

Włącz synchronizację danych między LOGO! BM a chmurą z określoną częstotliwością.

Okres (d:h:m:s)

Interwał czasowy, w którym LOGO! BM synchronizuje dane z chmurą.

Uwaga: Maksymalny interwał czasowy to 23 dni: 23 godziny: 59 minut: 59 sekund.

Przy zmianie

Jeśli wartość się zmieni, możesz zaznaczyć to pole wyboru, aby zsynchronizować zmiany z chmurą.

Zapisywalne

Zapisz dane z chmury do LOGO! BM.

Format danych LOGO! Web Server

Wybierz format danych jako ciąg szesnastkowy.

Uwaga: Maksymalna całkowita liczba rekordów danych to 160 bajtów.

Format danych tablicy

Wybierz format danych jako tablicę liczb całkowitych.

Uwaga:

  • Maksymalna całkowita liczba rekordów danych to 160 bajtów;

  • Całkowita długość wszystkich danych w chmurze nie powinna przekraczać 160.

+

Dodaj wiersz.

-

Usuń wiersz.

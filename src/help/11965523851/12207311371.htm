Plik -> Właściwości

Plik -> Właściwości, gdy okno projektu jest aktywne

Jeśli wybierzesz polecenie menu Plik -> Właściwości, gdy okno projektu jest aktywne, LOGO! wyświetli dialog "Właściwości przeglądu projektu". W tym oknie dialogowym możesz skonfigurować właściwości, takie jak Nazwa projektu, Autor i Klient. W polu Komentarz możesz wprowadzić opis projektu lub notatki związane z projektem. W polu "Hasło projektu sieciowego" możesz ustawić hasło dla projektu sieciowego lub zmodyfikować hasło projektu sieciowego.

Uwaga

Zasady dotyczące hasła projektu sieciowego są następujące:

  • Maksymalna długość hasła wynosi 10.

  • Obsługiwane są znaki ASCII.                       

Plik -> Właściwości, gdy edytor schematów jest aktywnym oknem

Jeśli wybierzesz polecenie menu Plik -> Właściwości, gdy edytor schematów jest aktywnym oknem, LOGO! wyświetli okno dialogowe "Ustawienia LOGO!". W tym oknie dialogowym możesz skonfigurować ustawienia offline i online dla LOGO!.

Możesz ustawić następujące parametry dla modułu podstawowego LOGO! w trybie offline:

  • Ogólne

  • Typ sprzętu

  • Ustawienia we/wy

  • Nazwy we/wy

  • Hasło programu

  • Włączenie zasilania

  • Tekst komunikatu

  • Dodatkowe informacje

  • Statystyki

  • Komentarz

  • Przestrzeń adresowa Modbus

  • Ustawienia zdarzeń e-mail

Możesz ustawić następujące parametry dla modułu podstawowego LOGO! w trybie online:

  • Połącz z LOGO!

  • Pokaż wersję oprogramowania sprzętowego

  • Przypisz adres IP

  • Ustaw zegar

  • Tryb pracy

  • Wyczyść program i hasło

  • Ekran włączania TD

  • Licznik godzin

  • Prześlij dziennik danych

  • Diagnostyka

  • Czas letni/zimowy

  • Ustawienia kontroli dostępu

  • Dynamiczny filtr IP serwera

  • Synchronizacja zegara z przełącznikiem EM

  • Ustawienia NTP

  • Ustawienia transferu danych w chmurze

  • Ustawienia połączenia z chmurą

  • Ustawienia certyfikatu

  • Ustawienia e-mail

Narzędzia -> Transfer -> Zatrzymaj LOGO!

LOGO!Soft Comfort wymaga potwierdzenia interfejsu połączenia przy każdej próbie komunikacji. Szczegółowe informacje znajdziesz w "Narzędzia -> Transfer".

→ Narzędzia → Transfer → Zatrzymaj LOGO!

Po kliknięciu tego symbolu zmienisz tryb podłączonego LOGO! z trybu RUN na STOP.

Alternatywnie możesz również kliknąć przycisk na Standardowym pasku narzędzi.

Uwaga

Ta funkcja jest dostępna tylko w urządzeniach serii sprzętowej 0BA6 lub nowszych.

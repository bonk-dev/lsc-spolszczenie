Plik -> Właściwości - Ogólne

W tej zakładce możesz ustawić nazwę i IP dla swojego urządzenia. Możesz wprowadzić adres, maskę podsieci i bramę, aby przypisać IP do modułu bazowego LOGO!. Możesz skonfigurować połączenie Ethernet dla swojego urządzenia.

W polu Nazwa programu możesz wprowadzić nazwę programu o długości do 16 znaków dla programu obwodu. Po przesłaniu, wyświetlacz na pokładzie LOGO! pokazuje program obwodu pod tą nazwą.

W polu Połączenia Ethernet możesz włączyć dostęp S7 lub dostęp Modbus oraz skonfigurować połączenie Ethernet.
